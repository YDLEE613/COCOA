Problem:
---
Bob loves numbers with 4 and 7 only. Write a program that print kth smallest number that Bob loves.

Input:
---
First line: K (1<=K<=10^9)


Output:
---
First line: kth smallest number

Example:
--
(1)  
_Input:_  
1

_Output:_  
4

(2)  
_Input:_  
2  

_Output:_  
7

(3)  
_Input:_  
3  

_Output:_  
44
